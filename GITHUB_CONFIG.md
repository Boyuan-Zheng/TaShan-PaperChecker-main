## About 描述

```
PaperChecker 是一个基于 AI 的学术论文引用检查工具，由中国科学院大学他山学科交叉创新协会 Agent4S 项目组开发。能够自动解析 Word 和 PDF 文档，提取引用和参考文献，进行智能匹配分析。完整支持作者-年份格式（中英文），识别缺失引用、年份不一致和格式问题，生成详细检查报告，帮助研究人员提高学术写作质量和规范性。
```
